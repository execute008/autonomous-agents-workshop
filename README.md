## 🚀 Quick Start

### OpenClaw (Cloud / Server)
```bash
npm install -g openclaw
openclaw configure
openclaw gateway start
```

### ZeroClaw (Local / Private)
- Download from https://www.zeroclawlabs.ai
- Runs fully offline — no data leaves your machine
