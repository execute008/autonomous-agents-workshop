# HEARTBEAT.md — Example Periodic Checklist

## Every heartbeat
- [ ] Check for new GitHub notifications (if gh CLI configured)
- [ ] Any calendar events in the next 2 hours?

## Daily (once per day)
- [ ] Update memory/YYYY-MM-DD.md with session summary
- [ ] Review MEMORY.md — anything outdated to prune?

## Weekly
- [ ] Run `git status` on active projects
- [ ] Check for skill updates: `npx clawhub update --all`
